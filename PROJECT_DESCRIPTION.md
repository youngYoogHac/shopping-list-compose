# Shopping List

## Описание проекта

Приложение представляет собой список покупок, реализованный на Kotlin с использованием Jetpack Compose.

Основные возможности:

- добавление товара;
- удаление товара;
- сохранение списка покупок после перезапуска приложения;
- переключение светлой и тёмной темы;
- сохранение выбранной темы после перезапуска приложения.

---

# Архитектура проекта

Проект построен по паттерну MVVM.

Схема взаимодействия компонентов:

ShoppingScreen → ViewModel → Repository → Room/DataStore

---

# Структура проекта

## data

Слой данных приложения.

### datastore

Хранение настроек приложения.

#### DataStoreExt.kt

Создаёт экземпляр DataStore для хранения настроек.

---

### local

Работа с локальной базой данных Room.

#### ShoppingItemEntity.kt

Описывает таблицу покупок.

#### ShoppingDao.kt

Содержит методы для работы с базой данных.

- получение списка покупок;
- добавление покупки;
- удаление покупки.

#### ShoppingDatabase.kt

Главная база данных приложения.

---

### repository

Посредник между ViewModel и источниками данных.

#### ShoppingRepository.kt

Интерфейс работы со списком покупок.

#### ShoppingRepositoryImpl.kt

Реализация репозитория через Room.

#### ThemeRepository.kt

Интерфейс работы с настройками темы.

#### ThemeRepositoryImpl.kt

Реализация репозитория через DataStore.

---

---

### theme

#### ThemeViewModel.kt

# Работа приложения

## Сохранение покупок

Данные сохраняются в Room Database.

После перезапуска приложения список покупок восстанавливается автоматически.

---

## Сохранение темы

Состояние переключателя темы сохраняется в DataStore.

После повторного запуска приложения выбранная тема остаётся активной.

---

# Точка входа

## MainActivity.kt
